Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le World Wide Web, abrégé en WWW, W3, le Web, la toile mondiale ou simplement la toile, est un système hypertexte public fonctionnant sur Internet. Le Web permet de consulter, à l'aide d'un navigateur, des pages regroupées en sites

Internet

Internet est le système mondial de réseaux informatiques interconnectés qui utilise la suite de protocoles Internet (TCP/IP) pour communiquer entre les réseaux et les appareils

Cloud

Le cloud computing est la disponibilité à la demande de ressources informatiques (telles que le stockage et l'infrastructure) en tant que services sur Internet

W3C

Le W3C ou World Wide Web Consortium désigne un organisme international à but non lucratif. Son rôle est de définir les standards techniques liés au web. Surtout, ils déterminent les mêmes règles pour tous les développeurs du monde.

Partie 2 - Distinguez les langages du Web

Langages client

Un langage client est un langage de programmation exécuté sur l’ordinateur du client, c’est-à-dire que le code est reçu puis exécuté sur l’ordinateur de la personne qui exécute, par exemple, une page web. Le HTML, CSS et Javascript sont des technologies client.

Navigateurs

Un navigateur web (web browser en anglais) est un logiciel permettant d'accéder à une page web et de l'afficher sur l'écran de l'utilisateur.

Langages serveur

Un langage serveur est un langage de programmation exécuté sur un serveur (ou ordinateur distant), c’est-à-dire que seul le résultat de l’exécution sera envoyé à l’ordinateur client.

Frameworks

Un framework en informatique – ou infrastructure logicielle – est un ensemble de bibliothèques, d'outils, de conventions de codage, et de modèles de conception qui simplifient et accélèrent le processus de développement logiciel. Il fournit une structure sur laquelle les développeurs peuvent construire leurs applications sans avoir à réinventer chaque composant à partir de zéro.

Bases de données

Une base de données est un ensemble organisé d'informations structurées, ou données, généralement stockées électroniquement dans un système informatique .

Site responsive

Un site responsive est un site qui est conçu et développé de façon à pouvoir s'adapter à toutes les résolutions d'écran. C'est donc un seul et même site qui peut être consulté sur ordinateur, sur smartphone ou sur tablette.

Application mobile

Une application mobile, est un logiciel développé exclusivement pour les smartphones et tablettes. Elles s'exécutent sur les appareils mobiles, via leur systèmes d'exploitation.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Un serveur est un ordinateur ou un système qui met des ressources, des données, des services ou des logiciels à la disposition d'autres ordinateurs, qualifiés de « clients », sur un réseau.

Adresse IP

Une adresse IP ( Internet Protocol )est une adresse unique qui identifie un appareil sur Internet ou un réseau local.

Nom d’hôte et DNS

Le nom d'hôte (ou hostname en anglais) est la désignation unique d'un ordinateur ou d'un nœud de réseau au sein d'un réseau. Les noms d'hôte sont des noms ou des chaînes de caractères spécifiques qui renvoient à un hôte et le rendent utilisable pour le réseau et les internautes

Le DNS, ou Système de Noms de Domaines, est un système essentiel à la navigation sur Internet. Il traduit les noms de domaine en adresses IP, permettant ainsi aux utilisateurs d'accéder aux sites web sans avoir à mémoriser une suite complexe de chiffres.

Partie 4 - Identifiez les métiers du web

Développeurs

Comme son nom l’indique, le développeur web conçoit et développe les sites, plateformes et applications que l’on retrouve sur Internet.

Product Managers et chefs de projets

Le Product Manager (PM) a une fonction stratégique : il organise le cycle de vie complet de son produit, de sa conception à sa mise sur le marché et au suivi de son utilisation. Il travaille donc sur un planning au temps long et mène les différentes étapes de développement.

Designers

À la fois artiste et informaticien, le webdesigner est capable de réaliser une interface web ergonomique et un design adapté au contenu d'un site Internet donné.

Testeurs

La Quality Assurance (QA) ou Assurance Qualité est un ensemble de processus et de pratiques utilisés pour garantir la qualité d’un produit ou d’un service. Elle vise à s’assurer que les normes de qualité sont respectées tout au long du cycle de vie du produit, depuis sa conception jusqu’à sa livraison aux clients.

Techniciens informatique

Polyvalent, le technicien de maintenance en informatique est capable de faire face à de nombreux problèmes. Et ce, avec le plus de rapidité et d'efficacité possible.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

Définition

CSS (Cascading Style Sheets, ou en français, feuilles de style en cascade) est un langage qui permet de définir l’apparence de nos éléments HTML. Les web designers utilisent le langage CSS pour réaliser le design d’un site.

JavaScript

Front-End

Définition

Le JavaScript est un langage de programmation principalement utilisé pour rendre les pages web interactives. C’est l’un des langages fondamentaux du web, avec HTML (structure) et CSS (style).

JavaScript

Back-End

Définition

Traditionnellement, JavaScript était utilisé uniquement dans le navigateur (front-end) pour rendre les pages web interactives. Mais depuis l’arrivée de Node.js, il est aussi utilisé côté serveur, c’est-à-dire en back-end.

PHP

Back-End

Définition

PHP (Hypertext Preprocessor) est un langage de programmation côté serveur, principalement utilisé pour créer des sites web dynamiques. Il a été créé en 1994 par Rasmus Lerdorf et reste très utilisé aujourd’hui, notamment dans les systèmes de gestion de contenu (CMS) comme WordPress, Drupal ou Joomla.

Java

Back-End

Définition

Le Java est un langage de programmation orienté objet, créé en 1995 par Sun Microsystems (racheté ensuite par Oracle). Il est aujourd’hui l’un des langages les plus utilisés au monde, notamment pour des applications d'entreprise, web, mobiles (Android), embarquées et back-end.

Python

Back-End

Définition

Python est un langage de programmation de haut niveau, facile à lire et à écrire, utilisé pour créer des applications web, des scripts, des programmes scientifiques, des outils d’automatisation, de l’intelligence artificielle, et bien plus encore.Il a été créé par Guido van Rossum dans les années 1990, et est aujourd’hui l’un des langages les plus populaires au monde.

C#

Back-End

Définition

Le C# (prononcé "C sharp") est un langage de programmation moderne, orienté objet, développé par Microsoft au début des années 2000.

Bases de données

Définition

Une base de données est un système organisé de stockage d’informations, qui permet de conserver, retrouver, modifier et supprimer des données de manière structurée et efficace.

Boîte à outils du développeur

  • IDE comme VS Code : Un environnement de développement intégré (IDE) est une application logicielle qui aide les programmeurs à développer efficacement le code logiciel.
  • Github : GitHub est une plateforme en ligne qui permet de stocker, gérer et collaborer sur du code source à l’aide de Git, un système de contrôle de version.
  • Librairies / bibliothèques : Les bibliothèques informatiques, également appelées librairies, sont des éléments particulièrement utiles aux développeurs. Elles permettent de simplifier et de rationaliser la programmation
  • Frameworks : En génie logiciel et en programmation, un framework est un ensemble de composants logiciels réutilisables qui permettent de développer de nouvelles applications plus efficacement. La réutilisation du développement et de la recherche existants est un principe essentiel dans tous les domaines de l'ingénierie.
